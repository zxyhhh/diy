#### 1. Account & Login

**Q: How to register an account?**
A: OurLoop supports the following login methods:
- Google Account Sign-in
- Apple ID Sign-in
Your account will be created automatically on first login, no separate registration needed.


**Q: How to edit my profile?**
A: 
1. Go to homepage, tap your avatar in the top-right corner
2. In Account Settings, you can modify:
   - Display name
   - Avatar
   - Language preference

**Q: How to delete my account?**
A: 
1. Go to Account Settings page
2. Scroll to bottom, tap "Delete Account"
3. Confirm deletion
Note: After deletion, all personal data will be permanently removed and cannot be recovered.

#### 2. Task Management

**Q: How to create a task?**
A: 
1. Tap the "+" button in the bottom-right of homepage
2. Choose task type:
   - Personal Task: Only visible and completable by you
   - Shared Task: Shared within a group, completed collaboratively
3. Fill in task information:
   - Task name (required)
   - Emoji icon (optional)
   - Recurrence frequency
   - Reminder time
   - Assignment type (for shared tasks)

**Q: What recurrence frequencies are supported?**
A: OurLoop supports flexible recurrence settings:
- Daily
- Weekly (choose days of week)
- Monthly (choose date or nth weekday)
- Yearly
- Custom interval (e.g., every 3 days, every 2 weeks)

**Q: What assignment types are available for shared tasks?**
A: 
- **Rotate**: Assigned to members in rotation
- **Everyone**: All members need to complete
- **Free Choice**: Members choose to complete voluntarily

**Q: How to complete a task?**
A: 
1. Tap "Complete" button on task card
2. Optionally add:
   - Completion note
   - Completion photo
3. Tap confirm to submit

**Q: How to edit or delete a task?**
A: 
1. Tap task card to enter details page
2. Tap "â‹¯" menu in top-right
3. Select "Edit" or "Delete"

**Q: What is a task streak?**
A: Streak tracks consecutive days of task completion. Completing tasks continuously increases your streak count, motivating you to maintain habits. If interrupted, the streak resets to 0.

#### 3. Group Collaboration

**Q: How to create a group?**
A: 
1. Tap "Groups" tab in bottom navigation
2. Tap "Create Group"
3. Fill in group name and description
4. An invite code will be generated after creation

**Q: How to invite members?**
A: 
1. Enter group details page
2. Tap "Members" tab
3. Tap "Invite Members"
4. Share the invite code with others

**Q: How to join a group?**
A: 
1. Get the group invite code
2. Tap "Join Group" on groups page
3. Enter invite code
4. Confirm to join

**Q: How to view group activities?**
A: 
In the "Activity Log" tab of group details page, you can see:
- Task completion records
- Member joins/leaves
- Task creation/modifications

**Q: How to leave a group?**
A: 
1. Enter group details page
2. Tap "â‹¯" menu in top-right
3. Select "Leave Group"
Note: Group creators cannot leave, only delete the group.

#### 4. Statistics

**Q: How to view completion records?**
A: 
1. Tap "Stats" tab in bottom navigation
2. You can view:
   - Calendar view: Shows daily completion status
   - Monthly stats: Current month completion count
   - Total stats: Cumulative completion count
   - Completion list: Detailed completion history

**Q: What tasks are included in statistics?**
A: The stats page shows both:
- Personal task completions (green markers)
- Shared task completions (blue markers)

#### 5. Reminders & Notifications

**Q: How to set task reminders?**
A: 
When creating or editing a task, you can set:
- Reminder time (hour and minute)
- Days to remind in advance

**Q: Not receiving notifications?**
A: Please check:
1. System settings allow OurLoop to send notifications
2. In-app notification permission is enabled
3. Phone is not in Do Not Disturb mode

iOS users:
- Settings > Notifications > OurLoop > Allow Notifications

Android users:
- Settings > Apps > OurLoop > Notifications > Allow Notifications

#### 7. Data & Privacy

**Q: Where is my data stored?**
A: 
- Personal task data is stored on local device
- Shared tasks and group data are stored on cloud servers
- All data transmission uses HTTPS encryption

**Q: Will my data be lost?**
A: 
- Shared task data syncs to cloud in real-time, won't be lost
- Personal task data is only stored locally
- Recommend logging in regularly to ensure data sync

**Q: How to export data?**
A: Data export is not currently supported. We are developing this feature, stay tuned.

**Q: Where to view privacy policy?**
A: You can find "Privacy Policy" and "Terms of Service" links at the bottom of Account Settings page.

#### 8. Membership Subscription

**Q: What are the limitations of free version?**
A: Free version includes:
- Create unlimited personal tasks
- Join up to 2 groups
- Basic statistics

**Q: What are membership benefits?**
A: Membership includes:
- Create unlimited groups
- Join unlimited groups
- Advanced analytics
- Data export
- Priority technical support

**Q: How to subscribe?**
A: 
1. Go to Account Settings page
2. Tap "Upgrade to Premium"
3. Choose subscription plan (monthly/yearly)
4. Complete payment through App Store/Google Play

**Q: How to cancel subscription?**
A: 
iOS users:
1. Open Settings > Apple ID > Subscriptions
2. Find OurLoop
3. Tap "Cancel Subscription"

Android users:
1. Open Google Play Store
2. Account > Subscriptions > OurLoop
3. Tap "Cancel Subscription"

#### 9. Technical Issues

**Q: App crashes?**
A: 
1. Ensure you're using the latest version
2. Try restarting the app
3. Restart your phone
4. If problem persists, contact technical support

**Q: Data sync failed?**
A: 
1. Check network connection
2. Confirm you're logged in
3. Try logging out and back in
4. Check app permissions

**Q: Cannot complete task?**
A: 
1. Check network connection
2. Confirm task permissions (shared tasks require assignment)
3. Try refreshing the page
4. Restart the app

**Q: Cannot find a feature?**
A: 
- Ensure you're using the latest version
- Some features may require membership
- Check this documentation for feature descriptions
