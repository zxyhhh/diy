Welcome to MindInbox.

If you run into any issues while using the app, or if you have questions about features, subscriptions, accounts, or data sync, please contact us at:

- Support email: zxyhdream@gmail.com

To help us investigate faster, please include the following when possible:

- Your device model and OS version
- App version
- Clear steps to reproduce the issue
- Screenshots or screen recordings of any error message
- Your sign-in method (Apple / Google / Email, if applicable)

We can help with issues including, but not limited to:

- Sign-in or account access problems
- Subscription, billing, and restore purchase issues
- Upload failures for messages, images, audio, or files
- Sync issues or missing content
- Permission problems related to notifications, microphone, or photos
- App crashes, performance issues, or unexpected behavior

Basic troubleshooting steps:

1. Make sure you are using the latest version of the app.
2. Check whether your network connection is stable, and try again on both Wi-Fi and cellular data.
3. If the issue is related to sign-in, subscription, or syncing, try signing out and signing back in.
4. If the issue involves photos, microphone, or notifications, verify that the required permissions are enabled in system settings.
5. If the problem continues, send us the details by email.

We will review your request and respond as soon as possible.
